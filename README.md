## 📌 Overview
This project is a deep learning–based **Multimodal Sentiment Analysis System** that detects human emotions using both **facial expressions** and **audio signals** from video input.  
The system combines **Computer Vision** and **Speech Emotion Recognition** techniques to improve prediction accuracy.

The model is trained on the **RAVDESS dataset** and can recognize multiple emotions including:

- Neutral
- Calm
- Happy
- Sad
- Angry
- Fearful
- Disgust
- Surprised

---

## 🚀 Features
✔️ Facial Emotion Recognition using OpenCV  
✔️ Audio Emotion Analysis using Librosa  
✔️ CNN + LSTM Deep Learning Model  
✔️ Multimodal Fusion Architecture  
✔️ Streamlit Web Application  
✔️ Emotion Confidence Scores  
✔️ Real-Time Video Emotion Prediction
